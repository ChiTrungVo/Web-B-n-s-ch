# Website Bán Sách
Đồ án xây dựng website thương mại điện tử bán sách sử dụng Spring Boot và Thymeleaf.

# Công nghệ sử dụng
Backend: Java 21, Spring Boot 3.5.5

Frontend: Thymeleaf, Thymeleaf Layout Dialect, HTML/CSS

ORM: Spring Data JPA (Hibernate)

Database: MySQL

Lưu trữ ảnh: Cloudinary

Validation: Jakarta Validation

Build tool: Maven

Khác: Lombok, Spring Boot DevTools


# Tính năng chính
## Người dùng (Khách hàng)

Xem danh sách sách, tìm kiếm và lọc theo thể loại

Xem chi tiết sách (tác giả, giá, mô tả, hình ảnh)

Đăng ký / Đăng nhập tài khoản

Thêm sách vào giỏ hàng

Đặt hàng và theo dõi đơn hàng


## Quản trị viên (Admin)

Quản lý danh mục sách (thêm, sửa, xóa)

Quản lý đơn hàng

Upload hình ảnh sách lên Cloudinary

Quản lý người dùng
